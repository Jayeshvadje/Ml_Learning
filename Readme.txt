* Develop real-world ML projects

### 📅 Week 1: Foundations

* Python basics & environment setup
* NumPy fundamentals
* Pandas & dataset handling
* Data visualization (Matplotlib, Seaborn)
* Statistics for ML
* Linear algebra intuition
* 🔹 Mini Project: Data Cleaning & Exploratory Data Analysis (EDA)

---

### 📅 Week 2: Core Machine Learning

* Introduction to ML (AI vs ML vs DL, ML types)
* Linear Regression
* Logistic Regression
* k-NN & Decision Trees
* Model evaluation metrics
* Overfitting, underfitting, bias-variance tradeoff
* 🔹 Project: Regression & Classification models

### 📅 Week 3: Feature Engineering

* Data preprocessing techniques
* Feature scaling methods
* Encoding categorical variables
* Feature importance & correlation analysis
* Model improvement using better features
* Model tuning basics
* 🔹 Project: Feature-engineered ML model

---

### 📅 Week 4: Deep Learning Basics

* Neural network fundamentals
* Backpropagation intuition
* TensorFlow / PyTorch setup
* Training a simple neural network
* Loss functions & optimizers
* Deep learning vs classical ML
* 🔹 Project: Neural Network implementation

---

### 📅 Week 5: Capstone & MLOps

* ML lifecycle
* Model persistence
* MLOps basics & deployment overview
* 🔹 Capstone Project: End-to-end ML solution
